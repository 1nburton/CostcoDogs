# Costco Dogs

A simple game scaffold to publish to App Store / Google Play via Expo.

## Setup

1. Install Expo CLI: `npm install -g expo-cli`
2. Install dependencies: `npm install`
3. Run locally: `npm start`

## Build

- iOS: `eas build --platform ios`
- Android: `eas build --platform android`

## App Store Preparation

- Update `app.json` bundleIdentifier/package.
- Add App Store icons in `assets/app-icon.png` and `assets/splash.png`.
- Set metadata in App Store Connect and Google Play Console.

## App Store / Play Store Description

Costco Dogs is a fast-paced, fun arcade game where you swipe to catch iconic dogs before time runs out. Level up as you collect more pups, beat your high score, and enjoy cute animations and upgrading doggie roster unlocks.

- Easy tap gameplay
- Timer-based high-score chase
- Progressive levels
- Light, family-friendly theme
